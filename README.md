## 🐍 Contribution Snake

<div align="center">

![snake gif](https://github.com/WHITEDECVIL/WHITEDECVIL/blob/output/github-contribution-grid-snake-dark.svg)

</div>

> ⚠️ **Snake setup:** Go to your repo → Actions → enable the `Generate Snake` workflow. Add the file below at `.github/workflows/snake.yml`

<details>
<summary>📄 Click to view snake.yml workflow</summary>

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

</details>
